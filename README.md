# TODO React Native App

A simple **TODO app built with React Native** to manage tasks efficiently.  
This project includes a mobile frontend and optional backend integration for storing tasks.

---

## Features

- Add, edit, and delete tasks
- Mark tasks as completed
- Simple and intuitive UI
- Offline support (if backend not connected)
- Ready for backend integration (API-based persistence)

---

## Tech Stack

- **Frontend:** React Native
- **Backend:** Next.js
- **State Management:** React hooks / Context API (if used)
- **Storage:** AsyncStorage (for local persistence)

---

## Getting Started

### Prerequisites

- Node.js >= 24
- npm
- React Native Expo CLI

### Installation

1. Clone the repository:

```bash
git clone https://github.com/Thaakie/TODOReactNative.git
cd TODOReactNative
```
